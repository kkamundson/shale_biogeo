# Full code for the MAP-FRAC ASV-to-MAG linking tool
##### The tool can be executed via the online interface here: https://apps.gis.colostate.edu/MAP-FRAC-Database/

Alternatively, the code ASV_matching_MAGs_v5.R can be executed directly in the R console using the associated file shale_MAGS_978_v2.txt which provides the necessary metadata from the shale MAG database.
