## This github is a code repository for processing of the 209 metagenomes associated with Amundson et al, in prep (2026)

The directory 'MetaG_procesisng' provides all code for processing raw metagenomic reads to MAGs, curating the final MAG database of 978 genomes, taxonomic assignment and functional annotation of MAGs, and read mapping for relative abundance. 

The directory 'MAG_linking_tool' contains the R code and associated MAG database metadata file for linking the MAGs presented in this study to user-provided 16S rRNA gene sequencing datasets.
